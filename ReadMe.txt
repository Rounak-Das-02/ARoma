Use case
This app helps customers accurately visualize menu items on the table, making the presentation of restaurant menus engaging and more appealing.

Navigating throught the app
User at first explore the menu with high-quality images and descriptions. 
User can select items from the menu and enter AR mode to visualize food items on their table.


Project technologies
We have used Android's modern toolkit "Jetpack Compose" for building the native UI. We have implemented the AR funcionality using SceneView.

Works only on Android. To build the app, open the project in Android Studio and click on "Build".
